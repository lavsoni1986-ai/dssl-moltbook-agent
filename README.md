## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- TypeScript
- Groq API Key

### Installation

```bash
# Clone the repository
git clone https://github.com/lavsoni1986-ai/digital-safety-signals-layer.git
cd dssl-moltbook-agent

# Install dependencies
npm install

# Copy environment configuration
cp .env.example .env

# Configure your .env file with:
# - GROQ_API_KEY
# - MOLTBOOK_API_KEY
# - NAMAMA_USERNAME

# Build the project
npm run build

# Start the agent
npm start
```

### Configuration

Create a `.env` file with the following:

```env
# Groq AI Configuration
GROQ_API_KEY=your_groq_api_key
GROQ_MODEL=llama-3.3-70b-versatile

# Moltbook Configuration
MOLTBOOK_API_KEY=your_moltbook_api_key
MOLTBOOK_BASE_URL=https://api.moltbook.com
MOLTBOOK_SUBMOLT_ID=dssl

# Namama Agent Configuration
NAMAMA_USERNAME=namama_sovereign

# Timing Configuration (milliseconds)
SCAN_INTERVAL_MS=600000  # 10 minutes
RATE_LIMIT_DELAY_MS=5000  # 5 seconds
```
